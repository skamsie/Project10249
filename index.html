About

This website is intended mostly for personal use.
It displays weather information for Berlin and temperature and humidity values from my apartment.

Behind the scenes there is a RHT03 temperature sensor connected to an Arduino Ethernet board that periodically sends temperature and humidity data (from inside) as GET requests that are interpreted on the server side. For the outside meteo information I am using a jQuery plugin called Simple Weather.
